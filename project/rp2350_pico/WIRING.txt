================================================================================
  Pin Functions:
================================================================================

  MOSI (GPIO 19):  SPI Master Out Slave In - Data from Pico to display
  SCK  (GPIO 18):  SPI Clock - Generated by Pico
  CS   (GPIO 17):  Chip Select - Active LOW (0=selected, 1=idle)
  D/C  (GPIO 20):  Data/Command - 0=Command, 1=Data
  RST  (GPIO 21):  Reset - Active LOW (0=reset, 1=normal)
  BUSY (GPIO 22):  Busy Status - 1=Busy (display updating), 0=Ready

================================================================================
  Notes:
================================================================================

  1. VOLTAGE: The SSD1681 operates at 3.3V. Do NOT connect to 5V!
  
  2. CURRENT: Display power consumption:
     - Idle: <1 mA
     - Updating: ~20-30 mA
     - The Pico's 3.3V regulator can supply enough current

  3. PIN CUSTOMIZATION: Pins can be changed in:
     project/rp2350_pico/driver/inc/rp2350_pico_driver_ssd1681_interface.h

  4. SPI PERIPHERAL: This uses SPI0. To use SPI1:
     - Change SSD1681_SPI_PORT to spi1
     - Update MOSI/SCK pins to SPI1 pins (GP11/GP10 or GP15/GP14)

  5. WIRE LENGTH: Keep wires short (<15 cm) for reliable 4 MHz SPI
     For longer cables, reduce SSD1681_SPI_BAUDRATE in the header file

================================================================================
  Typical Display Module Pinouts:
================================================================================

  Many SSD1681 modules use this common pinout:
  
    1. VCC   - 3.3V power
    2. GND   - Ground
    3. DIN   - MOSI (data in)
    4. CLK   - SPI clock
    5. CS    - Chip select
    6. DC    - Data/Command select
    7. RST   - Reset
    8. BUSY  - Busy flag

  VERIFY with your display's datasheet before connecting!

  Breadboard-Friendly Layout:

  If using a breadboard, this physical layout minimizes wire crossing:

    Display Side          Pico Side
    ┌─────────┐          ┌─────────┐
    │ 1  VCC  │◄─────────┤ 3V3     │
    │ 2  GND  │◄─────────┤ GND     │
    │ 3  DIN  │◄─────────┤ GP19    │
    │ 4  CLK  │◄─────────┤ GP18    │
    │ 5  CS   │◄─────────┤ GP17    │
    │ 6  DC   │◄─────────┤ GP20    │
    │ 7  RST  │◄─────────┤ GP21    │
    │ 8  BUSY │◄─────────┤ GP22    │
    └─────────┘          └─────────┘

================================================================================
  Alternative Pin Assignments:
================================================================================

  If the default pins conflict with your project, here are good alternatives:

  Option 1 (SPI0, different GPIOs):
    MOSI: GP3, GP7, GP19
    SCK:  GP2, GP6, GP18
    CS:   Any GPIO
    D/C:  Any GPIO
    RST:  Any GPIO
    BUSY: Any GPIO

  Option 2 (SPI1):
    MOSI: GP11, GP15
    SCK:  GP10, GP14
    CS:   Any GPIO
    D/C:  Any GPIO
    RST:  Any GPIO
    BUSY: Any GPIO

  Remember to update the pin definitions in the header file!
